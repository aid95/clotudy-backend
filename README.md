# 클로터디 (Clotudy)

> 이 프로젝트는 2020-2021년도 계명대학교 캡스톤디자인을 통해 진행되었습니다.

## 프로젝트 목표

교내에서 사용 중인 교수학습지원시스템의 부족한 기능과 낮은 활용성을 개선하고 다양화되고 있는 수업 방식에 대응하는 기능을 구상하여 교육의 질 향상

## 사용 기술 스택

### Back-end

- Django
- Redis
- Docker
- PostgreSQL

### Front-end

> 웹 디자인의 경우 캡스톤디자인 지원금을 이용해 웹 템플릿을 수정 및 추가하여 사용하였습니다.

- Websocket
- Javascript

## 사용된 협업 관리 도구

- JANDI
- Github
- Zoom

## 서비스 배포 환경

- AWS EC2
